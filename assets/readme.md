## 阶段说明

- **讨论**：需求与技术方案还在对齐，产出在 `01-沟通过程.md`
- **设计**：方案已固化（含验收标准），正在或已完成 `02-设计.md`
- **实施规划**：设计定稿，正在或已生成 `03-实施.md` 步骤清单 + `04-测试.md` 用例集
- **实施**：正在按 `03-实施.md` 落地代码，每步对照 `04-测试.md` 验证
- **完成**：已交付
- **暂停（修改或回退中）**：实施中发现方案问题（缺口→修改；根本错误→回退），回到设计/讨论阶段重新对齐

## 使用提示

- 想继续某个任务，直接进入对应任务文件夹，先读 `00-任务状态.md`。
- 想新建任务，序号取本表最大序号 +1，建好子文件夹后回到本表新增一行。
